Anchor Tag Usage

Here is a brief description about Anchor tag in HTML.

  • We Use Anchor Tag to Make Hyperlinks
  • We must use href attribute to give link to what we want to get redirect if we tap on that Hyperlink
  • We can also make images clickable as of Hyperlinks

Image Hyperlink
